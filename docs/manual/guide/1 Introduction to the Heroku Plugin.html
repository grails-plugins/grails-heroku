1 Introduction to the Heroku Plugin

The Heroku plugin makes it easy to deploy Grails applications to Heroku.

The primary function of the plugin is that it is very Grails-aware and it will autoconfigure applicable cloud services using the Heroku environment. This includes the Hibernate DataSource and the Mongodb, Redis, RabbitMQ, and Memcached plugins.
